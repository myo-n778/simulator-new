# 車上鉛直投げ上げ

## 目的

車から鉛直上向きに投げた物体を、外部視点と車視点で比較する。

- 外部視点では、物体が車の水平速度を受け継ぐため斜方投射に見える。
- 車視点では、車と同じ水平速度で動く条件のとき鉛直投げ上げに見える。

## 調整項目

- 鉛直初速度
- 車の初速度
- 車の加速度
- 重力加速度
- 空気抵抗
- 風の速度
- 風による加速度
- 再生速度

## 表示

- 外部視点
- 車視点
- 2画面比較
- 軌跡
- 速度ベクトル
- 抵抗なし比較

## 物理モデル

地上系で物体と車を積分する。

- 車: `x_car'' = a_car`
- 物体: `x'' = a_drag_x + a_wind`, `y'' = -g + a_drag_y`
- 空気抵抗は、風に対する相対速度に比例する簡易的な二乗抵抗として扱う。

空気抵抗、風、車の加速度を 0 にした標準状態では、車視点の横ずれがほぼ 0 になる。
