# 等速円運動と正弦波

## 目的

等速円運動する点の `y` 座標が、正弦進行波の変位と対応することを可視化する。

## 表示

- 上段左に等速円運動を表示する。
- 上段右に `y = 0` を基準軸とする横波を表示する。
- `x = 0` の `y` 軸は常時表示する。
- 下段に同じ位相を左右方向の粒子変位として扱う縦波表示を置き、横波の正弦波線も重ねる。
- 位相マーカーは横波と縦波で同じ `x` 位置を示す。

## モデル

- 右向き横波: `y(x,t) = A sin(ωt + φ - kx)`
- 左向き横波: `y(x,t) = A sin(ωt + φ + kx)`
- `ω = 2π / T`
- `k = 2π / λ`
- `v = λ / T`
- 縦波は同じ位相を粒子の左右変位として表示する。
- 円運動の黄色い点線は振幅 `1.00` の基準円として扱う。

## 操作

- 再生、一時停止、リセット
- 振幅 `A`
- 周期 `T`
- 波長 `λ`
- 再生速度
- 初期位相 `φ`
- 位相マーカー位置
- 進行方向の切替
- ガイド線、円の軌跡、粒子の色、対応線の表示切替
