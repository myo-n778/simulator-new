# サイコロ出目シミュレーター notes

## Scope

- 停止時にも複数面が見える3D風のサイコロを表示し、出目を高速に記録する。
- サイコロ本体は立方体に見えるよう、前面を正方形に近く保ち、上面と側面で奥行きを見せる。
- 目の点は面に貼った丸ではなく、面の内側に収まる浅い色付きの穴として表示する。
- `1` の目は赤い浅い穴で表示し、他の目も大きすぎない自然なサイズで表示する。
- 開始時と終了時は約1秒の転がしアニメーションを表示する。
- 1〜6の出現回数と割合を、目標回数に向かって積み上がる棒グラフでリアルタイム表示する。
- ガルトンボードと同じ共通シェル、暗色パネル、統計表示の構成に寄せる。

## Controls

- `開始` / `停止`
- `+1`
- `リセット`
- 記録速度
- 目標回数
- 回数プリセット: 50 / 100 / 500 / 1000 / 5000 / 10000
- 理論値 16.7% の目安線

## Verification Notes

- 乱数は `Math.random()` による一様な1〜6の整数。
- 総試行が目標回数に達したら自動停止する。
- 高速記録時は記録処理と画面更新を分け、画面は約30fpsで更新する。
