# 定滑車・動滑車の仕事シミュレーター notes

## Scope

- 理想的な滑車として扱い、摩擦と滑車の質量は含めない。
- 定滑車、動滑車1つ、複数動滑車を比較表示する。
- 物体の上昇距離、紐を引く距離、必要な力、支える紐の本数、仕事の関係を同時に示す。
- 複数動滑車は簡略化したブロックアンドタックル表示で、動滑車数 `n` に対して支える紐を `2n` 本として扱う。
- 物体表示は木箱、実験用おもり、本の束、重い荷物、モルカー（シロモ）を選択できる。
- 2026-06-07 update: 支柱、梁、滑車、紐、物体、床面を疑似3D表現に変更した。物理モデルは変更しない。

## Physics Model

- 物体の重さ: `W = mg`
- 支える紐の本数: `N`
- 必要な力: `F = W / N`
- 物体を上げる高さ: `h`
- 紐を引く距離: `L = Nh`
- 仕事: `F L = W h`

## Out of Scope

- 滑車の質量
- 軸や紐の摩擦
- 紐の伸び
- 実験装置の厳密なロープ取り回し
