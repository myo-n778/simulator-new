# 相対速度シミュレーター notes

## Purpose

- 1次元と2次元の相対速度を同時に比較する。
- 第3者視点、当事者視点、上空からの視点を同時に表示し、基準を変えたときの見え方を確認する。

## Model

- AとBは等速直線運動をする。
- 1次元では、相対速度を `v_B - v_A` として表示する。
- 2次元では、相対速度ベクトルを `(v_Bx - v_Ax, v_By - v_Ay)` として表示する。
- 当事者視点では、選択した観測者を原点に固定し、相手の位置と速度を相対量として描く。
- 当事者視点では、上段にFPS風の主観ビューを置き、相手の表示サイズで近づく/遠ざかるを表す。
- 主観ビューの近づく/遠ざかる判定は、相対位置ベクトルの距離変化率から決める。
- 1次元の主観ビューでは左右方向へ流さず、画面中央の奥行き方向で「奥から手前」「手前から奥」を表す。
- 1次元の主観ビューでは、相対位置が0をまたいだときに観察者が振り返ったものとして、見ている向きの表示、背景色、短い反転演出を切り替える。

## UI

- 1次元モードと2次元モードを左右に同時表示する。
- 各モードに次の3視点を置く。
  - 第3者視点
  - 当事者視点
  - 上空からの視点
- 当事者視点は、上段を主観ビュー、下段を読み取り用の相対座標ビューとして同じカード内で上下に表示する。
- 操作は再生、一時停止、1コマ送り、リセット、観測者切替、軌跡、グリッド、速度スライダーを含む。

## Checks

- `public/simulators/relative-velocity/index.html` が単独で開ける。
- 共通シミュレータヘッダーが表示される。
- `PhysicsSimClock.FixedStepClock` で再生、一時停止、1コマ送り、リセットが動く。
- 1D/2Dの両方で第3者視点、当事者視点、上空からの視点が同時に表示される。
- 当事者切替で、相対速度の符号と表示対象が切り替わる。
- 当事者視点の主観ビューで、相手が近づくと大きく、遠ざかると小さく表示される。
- 1次元当事者視点の主観ビューでは、相手の横位置ではなく奥行き方向の距離変化として表示される。
- 1次元当事者視点で相手が目の前を通過すると、背景色と「+x/-x方向を見ている」の表示が切り替わり、振り返り演出が出る。
