# 相対速度シミュレーター notes

## Purpose

- 1次元と2次元の相対速度をモード切替で比較する。
- 各モード内で、第3者視点、当事者視点、上空からの視点を表示し、基準を変えたときの見え方を確認する。

## Model

- AとBは等速直線運動をする。
- 1次元では、相対速度を `v_B - v_A` として表示する。
- 2次元では、相対速度ベクトルを `(v_Bx - v_Ax, v_By - v_Ay)` として表示する。
- 2次元では、Aから見たBの相対速度を `v_AB = v_B - v_A` とし、速度の関係を `v_A + v_AB = v_B` として表示する。
- 2次元では表示範囲の端をつなげず、AまたはBが境界に達したら初期位置へ戻す。
- 当事者視点では、選択した観測者を原点に固定し、相手の位置と速度を相対量として描く。
- 当事者視点では、上段にFPS風の主観ビューを置き、相手の表示サイズで近づく/遠ざかるを表す。
- 主観ビューの近づく/遠ざかる判定は、相対位置ベクトルの距離変化率から決める。
- 1次元の主観ビューでは左右方向へ流さず、画面中央の奥行き方向で「奥から手前」「手前から奥」を表す。
- 1次元の主観ビューでは、相対位置が0をまたいだときに観察者が振り返ったものとして、見ている向きの表示、背景色、短い反転演出を切り替える。
- 2次元の主観ビューでは、横方向のずれを残しつつ、距離に応じた表示サイズと奥行き位置の変化を1次元より強めに出す。

## UI

- 1次元モードと2次元モードは上部のモード切替で表示を切り替える。
- モード切替に合わせて、表示するパラメータと読み取り欄も切り替える。
- 各モードに次の3視点を置く。
  - 第3者視点
  - 当事者視点
  - 上空からの視点
- 当事者視点は、上段を主観ビュー、下段を読み取り用の相対座標ビューとして同じカード内で上下に表示する。
- 操作は再生、一時停止、1コマ送り、リセット、観測者切替、軌跡、グリッド、速度スライダーを含む。
- 2次元では初期位置を点線の円と点線の移動経路で残し、現在位置の実体が動いていく形にする。
- 2次元上空視点では、`v_A`、`v_AB`、`v_B` の速度ベクトルを表示し、`v_A + v_AB = v_B` を確認できるようにする。
- 2次元で境界に達した場合は、ワープ表示ではなく時刻と軌跡を初期状態へ戻す。

## Checks

- `public/simulators/relative-velocity/index.html` が単独で開ける。
- 共通シミュレータヘッダーが表示される。
- `PhysicsSimClock.FixedStepClock` で再生、一時停止、1コマ送り、リセットが動く。
- 1D/2Dのモード切替で、表示されるパラメータ、読み取り欄、3視点が切り替わる。
- 当事者切替で、相対速度の符号と表示対象が切り替わる。
- 当事者視点の主観ビューで、相手が近づくと大きく、遠ざかると小さく表示される。
- 1次元当事者視点の主観ビューでは、相手の横位置ではなく奥行き方向の距離変化として表示される。
- 1次元当事者視点で相手が目の前を通過すると、背景色と「+x/-x方向を見ている」の表示が切り替わり、振り返り演出が出る。
- 2次元当事者視点では、遠ざかると相手の表示が小さく奥へ寄り、近づくと大きく手前へ寄る。
- 2次元では初期位置が点線で残り、現在位置が動く。
- 2次元上空視点で `v_A + v_AB = v_B` のベクトル関係が表示される。
- 2次元でAまたはBが境界に達すると、初期位置に戻り、ワープした軌跡が残らない。
